    Introduction

    This API documentation page for My Idea Pool API.

    Authentication

    Sign Up

    Required parameters for signing up includes name, valid email and password. A valid password must be at least 8 characters long and must contain at lease one UPPERCASE, one lowercase and one Number

    Sign Up :

      POST '/users'

    Request Body: 

      { name: 'John Doe', email: 'john@email.com', password: 'PassW0rd' }

    NB Password must be at least 8 characters long and must contain at lease one UPPERCASE, one lowercase and one Number

    Response: 

      status: 201

  { jwt: 'Valid JWT Access Token String', refresh_token: 'Valid Refresh Token String' }

    Login

    Required parameters for logging in is the refresh_token

    Login:

      POST '/access-tokens'

    Request Body: 

      {  email: 'john@email.com',  password: 'PassW0rd' }

    Response: 

      status: 201

  { jwt: 'Valid JWT Access Token String', refresh_token: 'Valid Refresh Token String' }

    Refresh Token

    Required parameters to get a new JWT access token in is the refresh_token

    Refresh Token:

      POST '/access-tokens/refresh'

    Request Body: 

      { refresh_token: 'Valid Refresh Token String' }

    Response: 

      status: 200

  { jwt: 'Valid JWT Access Token String' }

    Logout

    Required parameters for loggin out is the refresh_token

    Logout:

      DELETE '/access-tokens'

    Request Body: 

      { refresh_token: 'Valid Refresh Token String' }

    Response: 

      status: 204

  { }
